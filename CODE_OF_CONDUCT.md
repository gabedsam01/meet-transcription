# Code of Conduct

This project is committed to maintaining a respectful, constructive, and welcoming environment for everyone.

Contributors are expected to communicate with professionalism, assume good intent, and help keep discussions focused on improving the project.

Disagreements are welcome when handled with respect and technical clarity. Personal attacks, disruptive behavior, or repeated bad-faith interactions are not acceptable.

If you notice conduct that harms the project environment, please open a private report with the maintainer (see [SECURITY.md](SECURITY.md) for private contact channels).

The maintainer may moderate issues, pull requests, discussions, or contributions that do not align with these expectations.

Thank you for helping keep this project useful, respectful, and collaborative.
